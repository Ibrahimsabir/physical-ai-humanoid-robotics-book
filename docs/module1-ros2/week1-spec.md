# Week 1 Specification: ROS 2 Fundamentals

## Learning Objectives
- Understand ROS 2 architecture and communication patterns
- Create basic ROS 2 publisher and subscriber nodes
- Learn about topics, services, and actions in ROS 2

## Required Knowledge Entering Week 1
- Python proficiency
- Basic understanding of distributed systems concepts

## New Capabilities Acquired in Week 1
- Create ROS 2 publisher/subscriber nodes
- Understand message passing between nodes
- Implement basic communication patterns

## Deliverable Artifact Specification
- Basic ROS 2 workspace with communication nodes
- Publisher node that sends joint position data
- Subscriber node that receives and processes joint position data

## Chapter Contract
- Introduction to ROS 2 concepts and architecture
- Step-by-step guide to creating publisher/subscriber nodes
- Code examples with explanations
- Common pitfalls and best practices
