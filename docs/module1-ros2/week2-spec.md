# Week 2 Specification: Robot Modeling

## Learning Objectives
- Understand URDF (Unified Robot Description Format) for robot modeling
- Create kinematic models of humanoid robots
- Learn about transforms and coordinate systems in robotics

## Required Knowledge Entering Week 2
- Week 1 ROS 2 fundamentals
- Basic understanding of 3D geometry and coordinate systems

## New Capabilities Acquired in Week 2
- Define robot kinematics and visual properties using URDF
- Create joint definitions and kinematic chains
- Understand TF (Transform) coordinate system

## Deliverable Artifact Specification
- URDF robot model file with joint definitions
- Visual and collision properties for robot links
- Transform configuration for coordinate system

## Chapter Contract
- Introduction to URDF and robot description format
- Step-by-step guide to creating a simple robot model
- Explanation of links, joints, and transforms
- Code examples with URDF XML structure
- Best practices for robot modeling
